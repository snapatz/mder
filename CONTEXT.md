# Markdown Viewer

A desktop app for opening, reading, and editing Markdown documents with a polished rendered view as the primary experience.

## Language

**Markdown Document**:
A user-owned `.md` file opened by the app.
_Avoid_: Note, page

**Viewer**:
The primary reading surface that renders a Markdown Document as a finished HTML-like page.
_Avoid_: Preview

**Editor**:
The source editing surface for a Markdown Document.
_Avoid_: Text box, input

**Dual Pane**:
A mode that shows the Editor and Viewer at the same time.
_Avoid_: Split preview

**Markdown Profile**:
The Markdown syntax the Viewer promises to render correctly: CommonMark, GitHub Flavored Markdown, syntax-highlighted code blocks, and Mermaid fenced diagrams.
_Avoid_: Markdown flavor, renderer mode
